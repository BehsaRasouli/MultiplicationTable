# MultiplicationTable3

A simple C# Windows Forms application that generates a 10×10 multiplication table and displays it using a DataGridView.  
The table is automatically created when the form loads, using nested loops and a two-dimensional array.

## Features
- Generates a full 10×10 multiplication table  
- Displays data inside a clean, auto-sized DataGridView  
- Beginner-friendly code structure  
- Demonstrates usage of arrays, loops, DataTable, and WinForms UI components  

## How it Works
- A 2D integer array (`int[10,10]`) stores the multiplication values  
- A `DataTable` is created and filled dynamically  
- The DataGridView displays the generated table when the form loads  

## Requirements
- Visual Studio  
- .NET Framework / .NET Windows Forms  

## Usage
Just run the application — the multiplication table will appear automatically.
